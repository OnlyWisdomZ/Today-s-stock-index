今日股票指数项目，是统计国内大盘、国内板块和国外大盘、国外板块的股票的交易数据，比如股票代码，股票类型，名称，开盘价，收盘价，是涨停还是跌停，涨幅，成交量，成交金额等。通过xxl-job实现了每天9点半到11点半，下午一点到三点，也就是股票开盘时间内，对股票交易数据的采集，在每天的凌晨两点半对上一天所采集到的股票数据进行统计，并且同步地返回给前端页面进行展示。

技术上，使用的是Spring Boot框架和Mybatis框架，数据库层使用的Mysql，使用shading-jdbc进行分库分表，使用Redis做缓存，登录验证码使用的是redis和hutool，使用Rabbit MQ消息队列进行Redis缓存的刷新，easyExcel进行数据的导出，使用git进行代码的提交，另外使用Swagger进行api文档的生成。
